# Nexus - Attendance Tracker

A sleek, mobile-first PWA for tracking attendance at Xavier's College - Semester VI.

## Features

- **Dashboard**: View attendance percentage, bunk calculator, and quick stats
- **Daily Logger**: Mark attendance for each period with one tap
- **Calendar View**: Browse past attendance with color-coded indicators and college events
- **Syllabus Viewer**: Complete syllabus for all 5 subjects with priority markers
- **Settings**: Customize target percentage, enable reminders, export data

## Subjects Tracked

1. Indirect Taxation (IDTX)
2. Auditing and Assurance (AUDA)
3. Financial Reporting & FSA (FRFS)
4. International Business (INBA)
5. Ethics & Corporate Governance (ETCG)

## Tech Stack

- Pure HTML5, CSS3, JavaScript
- Font Awesome 6.5 icons
- Google Fonts (Outfit)
- LocalStorage for data persistence
- Progressive Web App ready

## Usage

Simply open `index.html` in your browser. The app works completely offline after first load.
